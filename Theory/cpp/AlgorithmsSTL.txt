Алгоритмы STL <algorithm>
(на примере массива)

- изменить, если вып условие
- удалить
- вызвать функцию, если вып усл
- аккумулировать
________________
- предикат - функция, служащая для проверки условия

bool p(int val) { return val < 2; } 

int c[3] { 1, 2, 3 }
int sum = sum_if(c, c + 3, p (- предикат)) - вернет сумму элементов массива, которые = 1
int ptr* = remove_if(c, c + 3, p) - удалит все элементы, которые = 1 со сдвиго сущ на место удаленных,
			и веренет указатель на начало очищенного участка памяти


Лямда функции:
[ ](int val) { return val < 2; } - лямбда-функция
тип возвращаемого значения определяется в соответствии с первым return
(т.е. типы возвращаемого значения у всех return должны совпадать )
sum_if(c, c + 3, [ ](int val) { return val < 2; });


При изменении структуры вектора его итераторы оказываются недейсвительными!
